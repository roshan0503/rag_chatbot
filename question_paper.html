How to Use Question Paper Builder
Load saved questions and build a printable question paper
1
Select Documents
Required
Select at least one Course — this is required before any documents will appear.
Optionally narrow down using Leg, Subject, Chapter, Topic, or Section — each level unlocks the next.
Use the Keyword Search bar to find documents by name. Click Search or press Enter to apply.
Click document cards to select them. Multiple selections are allowed. The selected count appears in the bar below the list.
Hover over a document name to see its full hierarchy path (Course → Section).
2
Filters & Total Marks
Optional
Question Type — Filter by MCQ, True/False, Fill in Blanks, Short Answer, or Descriptive. Leave on All Types to load everything.
Difficulty — Filter by Easy, Medium, or Hard. Multiple levels can be active at once. Defaults to All Levels.
Total Marks — Set the target marks for the paper using the slider or by typing a number. This can be changed at any time before generating the PDF.
Click Load Questions from Database to fetch matching questions from saved banks.
3
Review & Assign Marks
Step 3
All loaded questions appear in the table grouped by type. Use the filter bar at the top to show a specific type.
Use the Edit button to modify question text, options, correct answer, or marks for any row.
Use the Delete button to remove any question you do not need.
Assign marks to each question using the inline number input. The marks bar shows assigned vs. target — both must match before you can generate the PDF.
Watch the Assigned / Total bar — it turns green with ✓ Marks match! when the totals are equal. The PDF button is blocked until then.
Configure & Export
Final Step
Fill in the Paper Title, School / Institute (optional), and Duration (minutes).
Add optional per-line Instructions for students — each line becomes a separate instruction in the paper.
Check Include Answer Key to bind the correct answers at the end of the PDF.
Check Show Marks per Question to print the mark value alongside each question.
Download formats — Generate a formatted PDF question paper, or export the raw questions as TXT, Excel (.xlsx), or Word (.docx).
Hover away from this panel to close it.
