How to Use Question Generation
Follow these steps to generate your question paper
1
Select Documents
Step 1
Select at least one Course from the dropdown — this is required to see available documents.
Narrow down documents further by selecting Leg, Subject, Chapter, Topic, or Section — each level unlocks the next.
You can also use the Keyword Search bar to find documents by name or content — useful when you know what you're looking for.
Click any document card to select it. You can select multiple documents. Selected count is shown in the bar below the list.
Hover over a document name to see its full hierarchy path (Course → Section).
2
Pages, Difficulty & Marks
Step 2
Page Numbers — Optionally click a document row to set a specific page range (e.g. 1-5, 8, 10-12). Leave blank to scan the entire document.
If you enter a Topic Focus keyword in Step 3, page number selections are ignored — the AI will scan the full document for that topic instead.
Difficulty Level — Choose Easy, Medium, or Hard. This guides how the AI frames questions.
Total Marks — Use the slider or type a number (1–100). This sets the target marks for the entire paper. You will need to assign these marks to individual questions in the Results step.
3
Question Types & Topic Focus
Step 3
Select one or more question types: MCQ True/False Fill in Blank Short Answer Descriptive
For each selected type, set the number of questions using the + / − controls. At least one type must be active.
Topic Focus (optional) — Enter a specific topic like "Newton's laws" or "French Revolution" to direct the AI to focus questions on that subject within your selected documents.
When a Topic Focus is entered, page number ranges set in Step 2 have no effect — the entire document is searched for that topic.
Click Generate Questions once you are satisfied with all settings.
Review, Assign Marks & Export
Results
Assign Marks — Each question has an editable marks field. The sum of all question marks must equal the Total Marks set in Step 2 before you can generate a PDF.
The marks bar at the top of the results shows how many marks are assigned vs. the target. Aim for the ✓ Marks match! status.
Use the Edit button on any row to change the question text, options, answer, marks, or type.
Use the Delete button to remove any question you do not want.
Fill in the Paper Title, School / Institute, and Duration fields. Add optional per-line instructions for students.
Check options to Include Answer Key, generate a Separate Answer Sheet, or Show Marks Distribution inside the PDF.
Export formats — Download as PDF, TXT, Excel (.xlsx), or Word (.docx).
Save to Database — Click Save Questions to store all questions in the Question Bank for future reuse.
Hover away from this panel to close it.
