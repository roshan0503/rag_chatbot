How to Use Question Bank Explorer
Search, filter and manage your entire question library
1
Search by Keyword
Quick Start
Use the Search Question Text / Keyword field at the top to instantly find questions by any word in the question, options, answer, or generation keyword.
Press Enter or click Apply Filters to run the search. Matching text is highlighted in yellow across all results.
You can also search inside a specific Source Document — type the document name in the dedicated filter to narrow results to a single PDF source.
2
Apply Filters
Narrow Down
Question Type — Click one or more type pills (MCQ True/False Fill in Blank Short Answer Descriptive) to filter by question format.
Difficulty — Select Easy, Medium, or Hard pills to show only questions at that difficulty level. Multiple pills can be active at once.
Course — Pick a course from the dropdown to show only questions linked to documents from that course.
Marks Range — Enter a Min and/or Max value to filter by how many marks each question carries.
Date Range — Use the From / To date pickers to show questions created within a specific time window.
Admin-Only Filters
Admins Only
Filter By Role — Admins and Super Admins see a smart Role dropdown. Select a role (Super Admin, Admin, Instructor, Student) to view only questions created by users with that role.
Created By (User) — Once a role is selected, a searchable User dropdown loads all users with that role. You can search by Full Name, Username, or IC Number using the mode pills inside the dropdown.
Instructors and students can only see questions from their own question banks. Role and User filters are not shown to them.
3
Review Active Filters
Filter Chips
Every active filter appears as a chip row below the filter grid. Click the ✕ on any chip to remove that single filter instantly without resetting everything else.
The badge next to the title (e.g. 3 filters active) shows you at a glance how many filters are currently applied.
Click Clear All in the chip row, or the Reset button, to remove all filters at once and return to the full question list.
4
View, Edit & Delete Questions
Actions
Each question card shows the type badge, difficulty badge, and marks at the top. MCQ options and their correct answer are shown inline.
Click the View button to open a full-detail modal — shows the question text, all options, correct answer, bank name, course, created-by, and all source documents the question was generated from.
Click Edit to modify the question text, options, correct answer, marks, or difficulty. Changes are saved directly to the database.
Click Delete to permanently remove a question. A confirmation prompt appears before deletion.
Edit and Delete are only available on questions from your own banks. Admins can edit and delete any question.
5
Sort, View & Paginate
Layout
Use the Sort by dropdown to order results by newest, oldest, marks (high/low), type, difficulty, or bank name.
Toggle between Grid and List view using the buttons in the toolbar. List view also supports column sorting by clicking the table headers.
Results are paginated at 24 questions per page. Use the page buttons at the bottom to navigate, or click a specific page number to jump directly there.
The toolbar always shows Showing X of Y questions — the first number reflects active filters, the second is the total loaded.
Hover away from this panel to close it.
